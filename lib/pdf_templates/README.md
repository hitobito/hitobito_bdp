# PDF Templates

This directory contains PDF templates used by the Export::Pdf functionality.

## efz_antrag.pdf

Template for the eFZ (erweitertes Führungszeugnis) application form.
This should be a PDF form with the following fields:

- mitglied_name
- mitglied_address
- mitglied_number
- mitglied_birthdate
- mitglied_city
- group_name
- group_address
- group_mail
- group_phone
- group_url
- efz_recipient_name
- efz_recipient_address
- date

The template file should be provided by the stakeholders and placed here as `efz_antrag.pdf`.
